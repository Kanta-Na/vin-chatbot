## 概要
このプロジェクトは、ワイン愛好家向けのChatBotアプリです。
OpenAI APIを活用し、将来的にはRAGやSpotify APIと連携して、以下のような体験を提供予定です。

## 開発目的
- ChatBot開発の技術習得
- RAG構成の実践
- エンジニア転職/サービス立ち上げのポートフォリオ作成

## 実装機能（初期）
- ユーザー入力に対するOpenAIの応答表示
- CLIまたは簡易UIでのチャットインターフェース

## 使用技術
